手機/平板使用

珠仔

即選號玩法。珠仔玩法是在80個號碼中選出1至5個號碼組合成一組進行的投注。玩家將選投注號碼與中獎號碼對照，不需按照順序，根據所選號碼與中獎號碼相符的個數多少，來確定相應的中獎級別。分為正賭、反賭、全中三種下法，見下例。

  1. 正賭、全中

    2. 例：投注者購買的是08，18，38，58這四個號碼為一個組合，且該期開獎號碼中包含08，18，38，58這四個數字，則視為投注"4中4"，若開出號碼中只有18，38，58則視為投注"4中3"，賠率以下表派彩。

    正賭賠率設置

    選5 選4 選3 選2 選1
    中單雙251 - - - - -
    中4 21 51 - - -
    中3 3 6 20.3 - -
    中2 0 2 2.3 10.3 -
    中1 0 0 0 0 3.3

    全中賠率設置

    選5 選4 選3 選2 選1
    中5 1150 - - - -
    中4 0 240 - - -
    中3 0 0 48 - -
    中2 0 0 0 12 -
    奇和偶0 0 0 0 0 -
  2. 反賭

    3. 例：投注者購買的是02，04，06，08，10，12，14，16，18，20，22，24這12個號碼為一個組合，且該期開獎號碼中全不包含，則視為贏。

    反賭賠率設置

    選12 選11 選10 選9 選8
    中0 27 - - - -
    中0 0 20 - - -
    中0 0 0 14 - -
    中0 0 0 0 10 -
    中0 0 0 0 0 6
