  • 大小

    开出的20个号码的总数值为游戏结果。当20个号码总和为810时，810以上为大，810以下为小。

    例：开彩号码为02、03、07、08、11、16、17、26、29、32、33、35、39、43、50、56、68、72、74、80，总数值是701，则小。

    下注"小"为赢，下注"大"为输。

    赔率设置
    大 810 小 投注大/小，结果810
    赔率 1.95 108 1.95 投注退回

  • 单双

    开出的20个号码的总数值为游戏结果。总数值为奇数，为单；总数值为偶数，为双。

    例：开彩号码为02、03、07、08、11、16、17、26、29、32、33、35、39、43、50、56、68、72、74、80，总数值是701，则单。

    下注"单"为赢，下注"双"输。

    赔率设置
    单 双
    赔率 1.95 1.95

  • 奇和偶

    在开出20个中奖号码当中，如奇数号码数目比偶数的数目多时，则为奇局，投注"奇"者中奖；如偶数号码数目比奇数的数目多时，则为偶局，投注"偶"者中奖；如奇数和偶数数目相同时(均为 10个)，则为和，投注"和"者中奖。

    例：开奖号码为02、03、07、08、11、16、17、26、29、32、33、35、39、43、50、56、68、72、74、80

    奇数： 03、07、11、17、29、33、35、39、43 (奇数数目9个)

    偶数： 02、08、16、26、32、50、56、68、72、74、80 (偶数数目11个)

    其中奇数9个，偶数11个，此局偶数多为偶局。下注"偶"为赢，下注"奇"为输。

    赔率设置
    奇 和 偶
    赔率 2.3 4.3 2.3

  • 上中下

    号码1至40为上盘号码，41至80为下盘号码。开出的20个开奖号码中，如上盘号码占多数时，此局为上盘局；下盘号码占多数时为下盘局；上盘号码和下盘号码在此局开出的数目相同时(各10个数字)，此局为和盘。

    例：开奖号码为02、03、07、08、11、16、17、26、29、32、33、35、41、43、50、56、68、72、74、80

    上盘 - 02、03、07、08、11、16、17、26、29、32、33、35 (上盘数目12个)

    下盘 - 41、43、50、56、68、72、74、80 (下盘数目8个)

    例子中12个上盘数值，8个下盘数值，此局为上盘局。下注"上"为赢，下注"下"为输。

    赔率设置
    上 中 下
    赔率 2.3 4.3 2.3

  • 珠仔

    即选号玩法。珠仔玩法是在80个号码中选出1至5个号码组合成一组进行的投注。

    玩家将选投注号码与中奖号码对照，不需按照顺序，根据所选号码与中奖号码相符的个数多少，来确定相应的中奖级别。

    分为正赌、反赌、全中三种下法，并且其中只有”正赌”玩法，有提供单一投注多重中奖机会，详见下例。

    • 正赌、全中：
      例：投注者购买的是08，18，38，58这四个号码为一个组合，且该期开奖号码中包含08，18，38，58这四个数字，视为投注“4中4”，若开出号码中只有18，38，58，则视为投注“4中3”，详见以下派彩赔率表。

      正赌赔率设置
      选5 选4 选3 选2 选1
      中5 251 - - - -
      中4 21 51 - - -
      中3 3 6 20.3 - -
      中2 0 2 2.3 10.3 -
      中1 0 0 0 0 3.3
      全中赔率设置
      选5 选4 选3 选2 选1
      中5 1150 - - - -
      中4 0 240 - - -
      中3 0 0 48 - -
      中2 0 0 0 12 -
      中1 0 0 0 0 0

    • 反赌：
      例：投注者购买的是02，04，06，08，10，12，14，16，18，20，22，24这12个号码为一个组合，且该期开奖号码中全不包含，则视为赢。

      反赌赔率设置
      选12 选11 选10 选9 选8
      中0 27 20 14 10 6

  • 大小单双过关

    开出20个号码的总和数值为游戏结果，分为“大单”、“小单”、“大双”、“小双”。

    总和数值大于810为“大”，总和数值小于810为 “小”；总和数值为奇数为”单”，总和数值为偶数为”双”。

    通过大小和单双组合产生“大单”、“小单”、“大双”、“小双”四种结果。

    例：开奖号码为01、02、05、09、10、17、18、26、28、33、34、35、41、43、50、57、68、73、76、80，总数值是706，总和小于810，并且是偶数为双，则为“小双”。下注“小双”为赢，下注其他为输。

    赔率设置
    大单 小单 大双 小双 总和数值结果810
    赔率 3.7 3.7 3.7 3.7 投注回退

  • 大小指数

    以开出20个号码的总和数值为游戏结果，810以上为大，810以下为小，810为不分大小。

    例：当20个号码总和数值是860时，即指数+50点，如投注"大"即为赢，再计算每点投注的金额，如设定为10元1点，即赢50点，可赢得500元，如投注”小”的话，相反会亏损500元，但如设定的上限为300元，即赢输的极限就只会到300元。

    赔率设置
    最小点距 最大点距 投注指数大/小，结果810 赢佣金%
    5 600 投注回退 5

  • 五行

    五行，即总和数值范围。开出20个号码的总和分在五个数段中，五个数段以金、木、水、火、土命名： 金(210~695)、木(696~763)、水(764~855)、火(856~923)和土(924~1410)。

    例：开奖号码为01、02、05、09、10、17、18、26、28、33、34、35、41、43、50、57、68、73、76、80，总数值是706，则数值段在(696~763)内，则开出是“木”。下注“木”为赢，下注其他的四个的五行值为输。

    赔率设置
    金 木 水 火 土
    赔率 9.2 4.6 2.4 4.6 9.2

  • 溷合过关

    玩家可选择任何地区的投注选项进行过关投注，溷和过关的赔率固定为相乘。

  • 不可溷合过关之投注项目

    • 不分开奖来源，大小指数皆不可串关。
    • 不同开奖来源，除大小指数其馀皆可进行串关。
    • 同一开奖来源，「珠仔」不可和其馀玩法选项串关。
    • 同一开奖来源，「大小」不可和「上下」、「大小单双」、「五行」串关。
    • 同一开奖来源，「单双」不可和「大小单双」串关。
    • 同一开奖来源，「上下」不可和「大小单双」、「五行」串关。
    • 同一开奖来源，「大小单双」不可和「五行」串关。
    • 同一开奖来源，「奇偶」不可与「大小单双」、「单双」串关。

  • 追号与倍投

    • 追号定义：将一注或一组号码进行两期或两期以上的投注。最多可进行连续10期。
    • 倍投定义：将目前的投注金额，以倍率的方式进行投注，由1.0为起始，最高为3倍。
    • 追号终止条件：
      • 当达到玩家指定的追号期数，或设定的终止条件时。
      • 当帐户馀额不足于购买的投注项目时。
      • 当下一期追号注单，预期返还金额超过当期最大获利金额时。
      • 当购买的注单未进行开奖时。
      • 玩家手动停止追号时。

    • 追号起始时间点：追号从当前需要追号的订单开始，作为第一期追号，直到符合玩家设定的终止条件时停止追号。

      例：独赢 北京 大小【大】 投注金10元， 追号10期，倍投2倍

      期数 开奖结果 赔率 注金 结果
      第一期 北京 大小【小】 1.95 10 0
      第二期 北京 大小【小】 1.95 20 0
      第三期 北京 大小【小】 1.95 40 0
      第四期 北京 大小【大】 1.95 80 156
      第五期 第四期开奖赢，故系统自动停止后续注单追号
    • 其他说明：
      • 长龙提示：任何一种玩法中，开奖连续超过4次以上相同时，以”旗帜”标示。
      • 取消注单：当开奖来源未公布某期开奖结果时，如在1小时之后仍未更新，我们会斟酌进行取消该期投注单。
      • 快速开奖盘口：我们从欧洲博彩公司，引进了两组先进的自然机率开奖系统，分别为75秒、90秒开奖一次，并且开奖系统经过菲律宾博彩执照公司的审核监控，保证公平公正，提供您更丰富有趣的投注选择。
