赚取佣金 轻松盈利

如何赚取佣金？

当您注册成博博的合营伙伴后，基于您每月推荐到博博体育博彩和娱乐场的玩家所产生的盈利，通过您的用户名，我们可以追踪到流量、下载量、注册数量、有参与投注的会员数量等信息和所有的数据来计算佣金。 通过我们的市场营销工具，您的合营事业将会很容易建立起来。

只需4步骤，立即开启享受精彩人生：

  1. 注册账号后，申请开通代理资格。
  2. 联系代理专员，完善个人信息。
  3. 洽谈合作细节，获取专属佣金分红计划。
  4. 开通完成，即刻推广，招揽下线，赚取财富。
1.分层佣金结构

佣金计算范围：体育，真人娱乐场，游戏，基诺，棋牌等等。

活跃玩家（人) 净输赢 佣金比例
5-14 0 < 净输赢 < 200000 30%
15-30 200000 ≤ 净输赢 < 500000 35%
>30 ≥ 500000 40%

同时达到活跃会员和净输赢条件才能享受相应的佣金占成活跃会员定义：合作月内，代理下线会员有任何一项存款、提款、转账、投注行为的客户都算活跃使用者 。
*对于新注册代理账号，如其有疑似操纵行为，本公司保留3个月观察期之权利。

佣金结算规则

  1. 合作伙伴每月的佣金提成为正值时，等于或大于5个活跃会员才能启动领取佣金。乙方每月的佣金提成为负值时，不会启动领取佣金，所有负值金额会累积计算（与活跃客户数无关）。
  2. 当乙方正式成为甲方合作伙伴之日起算，直到当月最后一日，为合作伙伴运营收益计算时间段。之后按每月计算收益。
  3. 每月5-9号系统将自动生成上个月代理佣金账单。10-19号为对账时间。如代理对佣金账单无异议，每月20号可以申请提款上一个月的佣金。（1月份佣金，2月5-9号生成佣金账单，2月10日-2月19日为对账时间。2月20号，代理可登陆平台申请提取佣金。）
  4. 合作伙伴可根据帐户显示的提款余额申请提款，每月申请提款额度需大于或等于100RMB。如余额不足100RMB，将会累积到100RMB才能提取自动申请领取。
  5. 如当月的收入为负值，佣金则不产出，累计之下月可计算出正值佣金为止。如当月的结算为盈利，及能达到提款要求佣金将在解锁后，乙方即可申请领取。
  6. 合作伙伴不得操控下线玩家任何操作行为，否则将取消合作资格并锁定佣金。
    * 批量注册下线会员数量以达到活跃用户要求。
    * 操纵下线会员投注行为。
    * 认定有套取佣金的任何行为。
